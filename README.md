# Foodipedia

## What is it?

This app is provide to implement mealsDB API with kind of some features such as:

- List of category food
- List food by category
- Detail food with trailer video

## Tech stack

- Single activity using jetpack nav component
- Splashscreen API
- MVVM
- Hilt
- Flow
- LiveData
- Retrofit
- Moshi
- Youtube player
